# Vela

A browser-based, computer-vision-powered form coach for the Big 3.
MediaPipe runs in your browser to track joints in real time on live
camera or uploaded video; a Backboard-hosted Claude agent personalizes
the cues, the rules, and the next-session prescription based on your
training history.
